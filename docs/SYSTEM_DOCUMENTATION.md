# 货币基金套利提示系统文档

## 1. 系统概述

货币基金套利提示系统是一个用于监控和分析货币基金套利机会的系统，主要功能包括：

- 实时获取货币基金的价格、净值和收益率数据
- 计算货币基金的套利收益率
- 提供套利机会的提醒和通知
- 提供数据可视化界面，方便用户查看和分析数据

## 2. 系统架构

系统采用前后端分离架构：

- **后端**：使用Python Flask框架开发，负责数据获取、存储和计算
- **前端**：使用Vue 3 + Element Plus开发，负责数据展示和用户交互
- **数据库**：使用SQLite数据库存储数据

## 3. 安装说明

### 3.1 环境要求

- Python 3.7+
- Node.js 14+
- npm 6+

### 3.2 后端安装

1. 进入后端目录：
   ```bash
   cd backend
   ```

2. 创建虚拟环境：
   ```bash
   python -m venv venv
   ```

3. 激活虚拟环境：
   - Windows：
     ```bash
     venv\Scripts\activate
     ```
   - Linux/Mac：
     ```bash
     source venv/bin/activate
     ```

4. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

### 3.3 前端安装

1. 进入前端目录：
   ```bash
   cd frontend
   ```

2. 安装依赖：
   ```bash
   npm install
   ```

## 4. 使用说明

### 4.1 启动系统

1. 启动后端服务：
   ```bash
   python api.py
   ```

2. 启动前端服务：
   ```bash
   npm run dev
   ```

3. 或者使用一键启动脚本：
   ```bash
   python start_system.py
   ```

### 4.2 访问系统

- 前端页面：http://localhost:5173
- 后端API：http://localhost:5000/api

### 4.3 系统功能

1. **综合数据**：显示货币基金的综合数据，包括净值、买入价、买入收益率、卖出价、卖出收益率
2. **基金列表**：显示系统监控的货币基金列表
3. **价格数据**：显示货币基金的价格数据，包括最新价、买入价、卖出价、价格日期
4. **净值数据**：显示货币基金的净值数据，包括净值、净值日期
5. **收益率数据**：显示货币基金的收益率数据，包括收益率、买入收益率、卖出收益率、收益率日期
6. **提醒记录**：显示系统生成的套利提醒记录
7. **错误记录**：显示系统运行过程中产生的错误记录

### 4.4 数据刷新

点击各页面的"刷新数据"按钮可以手动刷新数据。

## 5. 维护说明

### 5.1 数据库维护

- 数据库文件：`backend/fund_arb.db`
- 定期备份数据库文件，以防止数据丢失

### 5.2 日志管理

- 调度器日志：`backend/scheduler.log`
- 定期查看日志文件，了解系统运行状态

### 5.3 数据更新

- 系统会自动定时更新数据，默认每10分钟更新一次
- 可以修改`scheduler.py`文件中的定时任务配置

### 5.4 系统监控

- 定期检查系统运行状态，确保数据获取和计算正常
- 检查API接口是否正常响应

## 7. 故障排除

### 7.1 后端服务无法启动

- 检查Python环境是否正确安装
- 检查依赖是否安装完整
- 检查端口5000是否被占用

### 7.2 前端页面无法访问

- 检查Node.js环境是否正确安装
- 检查依赖是否安装完整
- 检查端口5173是否被占用

### 7.3 数据无法更新

- 检查网络连接是否正常
- 检查数据源是否可访问
- 检查调度器是否正常运行

### 7.4 数据显示异常

- 检查数据库连接是否正常
- 检查数据获取是否成功
- 检查数据计算是否正确
